<link> For linking in an html document, usually for stylesheets. Place in head section. Needs "rel" value (ie. stylesheet) and "href" value.
<link> href="../resources/style.css" rel="stylesheet"</link>
<title> The title of your page, shows up in the browser tab. Place in head of document.
<title>Kayla's HTML & CSS Cheatsheet</title>
<a> Creates a link to another page (href='url') or a location in the current page (href='#').
<a href="www.codecademy.com">Visit This Site</a>
<a href="#contact">Contact Info</a>
